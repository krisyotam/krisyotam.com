# Code Block Display Testing

This file tests the display of code blocks using both native markdown syntax and the CodeBlock component.

## 1. Native Markdown Code Block with Filename

```javascript(example.js)
// This is a test JavaScript file with a long line that should wrap properly without affecting the line numbers column
function calculateSomethingComplex(input) {
  const result = input.map((item) => item * 2)
                       .filter((item) => item > 10)
                       .reduce((total, current) => total + current, 0);
  
  // This adds some horizontal scrolling to test the sticky line numbers
  console.log("This is a very long line of code that should cause horizontal scrolling to test if the line numbers remain visible and properly aligned when scrolling horizontally");
  
  return result;
}

// Multiple lines to test vertical scrolling
for (let i = 0; i < 10; i++) {
  console.log(`Line ${i + 1} of the code block`);
}
```

## 2. Native Markdown Code Block without Filename

```python
# Python code block without filename
def calculate_factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * calculate_factorial(n - 1)

# Test the function
for i in range(5):
    print(f"Factorial of {i} is {calculate_factorial(i)}")
```
